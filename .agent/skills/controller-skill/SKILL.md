# Controller Skill

本 Skill 定義了 Java Web 開發中的控制器 (Controller) 實作流程與最佳實踐。涵蓋傳統的 Servlet MVC 模式以及現代化的 Spring Boot RESTful API 開發架構。

## 目錄

- [Servlet MVC 控制器實作規範](./references/servlet-mvc.md)
- [Spring Boot RESTful 控制器實作規範](./references/rest-controller.md)

## 使用說明

根據專案需求選擇適合的架構：
1. **傳統 Web 應用 (SSR)**: 使用 [Servlet MVC](./references/servlet-mvc.md) 設計模式。
2. **現代前後端分離應用 (SPA)**: 使用 [Spring Boot RESTful](./references/rest-controller.md) 設計模式。
